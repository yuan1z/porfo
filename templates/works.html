Stochastic Computing-Based DNN Accelerator

Designed and optimized stochastic computing (SC)-based Deep Convolution Neural Network accelerator. Experimental results on LeNet and AlexNet with the ImageNet and MNIST dataset show low hardware footprints with tiny accuracy loss.

Require knowledge: Deep Learning, Matlab, Verilog, Physical Design Tool Flow

002/006

Emerging Cooling Models for Chips

Modeled emerging cooling technologies (e.g, Liquid cooling, two-phase cooling, and TEC) for High-perforamnce Chips. Also implemented a SPICE-based compact thermal simulator to achieve fast and accurate parallel thermal simulation for processors.

Require knowledge: C, Python, Linux scripting

003/006

MIPS 5 Stage Pipline CPU design

Designed the schematic and full-custom layout of a five-stage MIPS pipeline CPU with 1KB on-chip memory using Cadence Virtuoso and TSMC 180nm library. This CPU is capable of doing 16 different operations and additional hardware power gating and data gating methods are also included.

Require knowledge: Cadence Virtuoso, Perl Scripting

004/006

DDRIII Memory Controller Design

Designed a DDR-III memory controller using Verilog and TSMC 45nm library. The netlist is synthesized using Synopsis Design Compiler and the layout is generated using Cadence Encounter. Additional static timing analysis (STA) is evaluated using Synopsis PrimeTime.

Require knowledge: Verilog, Physical Design Tool Flow, Perl Scripting

005/006

Random Pulse Generator Design

Designed an on-chip random pulse generator (RPG) using CMOS LED, waveguide, and single photon avalanche diode (SPAD).

Require knowledge: Virtuoso, Verilog-A
